# LibTV 作者标签管理 - 快速预览

## 方式一：本地服务（推荐，100% 可用）

```bash
npm run preview:serve
```

然后打开浏览器访问：**http://localhost:4173**

## 方式二：直接打开 HTML 文件

双击 `preview.html` 或 `dist-preview/index.html` 在浏览器中打开。

> ⚠️ 部分浏览器对 `file://` 协议下的 ES 模块有限制，若出现白屏请改用方式一。

## 重新生成预览包

```bash
npm run build:preview
```

生成文件：
- `dist-preview/index.html` - 单文件（约 200KB，含全部 JS/CSS）
- `preview.html` - 同上，复制到项目根目录便于分享
